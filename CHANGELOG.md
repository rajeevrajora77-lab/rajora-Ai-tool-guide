## [1.0.0] — 2026-04-26

### Added
- Initial release of **Rajora.Ai Tool Guide**
- 150+ curated free developer tools across 6 progressive layers (L0–L5)
- Scroll-driven GSAP `ScrollTrigger` animations
- Interactive real-time search & filter directory
- Tool detail modals with install commands and docs links
- Dark neon theme (`#0B0C10` + `#B6FF2E`)
- Fully responsive layout (desktop, tablet, mobile)
- Keyboard navigation and reduced-motion accessibility support
- React 19 + TypeScript 5.9 + Vite 7 project scaffold
- Tailwind CSS 3.4 + Radix UI 53-component library
- MIT License, CONTRIBUTING guide, and `.gitignore`
